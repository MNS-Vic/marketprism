## 🏗 技术架构

### 后端技术栈
```yaml
语言: Python 3.12
框架: FastAPI + SQLAlchemy
数据库: Redis + ClickHouse
消息队列: Redis Pub/Sub
监控: Prometheus + Grafana
追踪: Jaeger
容器: Docker + Kubernetes
```

### 前端技术栈
```yaml
框架: Next.js 15 + React 19
UI库: Radix UI + Tailwind CSS
状态管理: React Hooks + Context
图标: Lucide React
类型: TypeScript
构建: Vite + SWC
```

### 部署架构
```yaml
容器化: Docker + Docker Compose
编排: Kubernetes
代理: Nginx
监控: Prometheus + Grafana
日志: ELK Stack (可选)
备份: 自动化备份脚本
```
